## Notes on source quality

- **Primary vendor docs** are weighted highest where they exist (DMG MORI, Renishaw, Universal Robots, ABB, KUKA, Yaskawa, Bystronic, Trumpf product pages and developer portals).
- **Standards-body docs** (MTConnect Institute, OPC Foundation, ISA, umati, ROS-Industrial) are primary for protocol details.
- **Integrator KBs and third-party guides** (MachineMetrics, Standard Bots, Vention, Tera Digital, TEKLYNX) are secondary — useful for cross-referencing claims that vendors don't make, but with the implicit understanding that they're often selling their own services. Cross-referenced with primary where possible.
- **Crowd-sourced pricing** (QVIRO, Standard Bots blog) is the only available data for robot prices because vendors don't publish — flagged as "industry estimate, not vendor-cited" wherever used.

## Notes on maintenance

This file should be retired or replaced when:
- A specific topic file graduates into a real ADR — that ADR's citations replace these.
- A specification major-version bump invalidates a primary spec link.
- A vendor radically restructures their docs (e.g., DMG MORI deprecates the IoTconnector page) — add a `## 2027 update` to the affected topic file with the new URL, keep the old one for history.

Treat fetch dates as authoritative. If a URL still resolves but the content has changed materially since the listed fetch date, re-research that topic before relying on the original claim.

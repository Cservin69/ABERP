#   * pre-serve BOOT openers that run BEFORE the shared Handle is constructed
#     (serve.rs Handle::open_default is the boot line) — record_upgrade_snapshot_mismatch_audit.
#
# Everything else the scanner flags is an in-serve write-fork that MUST route
# through the shared aberp_db::Handle. The migration worklist (the not-yet-migrated
# remainder) is tracked in tools/adr0099_write_fork_migration_worklist.txt.
